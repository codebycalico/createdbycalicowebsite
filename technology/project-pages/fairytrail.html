fairy trail uses webcam feed to detect a person or persons in the area and creates a particle system trail that follows them around. the code uses pose estimation and follows the first confident joing that it finds.

when no one is in the frame, it creates an explosion of particle systems like fireworks. this "idle" state continues until it detects someone in the frame. each time a person is detected, it assigns a random hue to the trail for that body.
